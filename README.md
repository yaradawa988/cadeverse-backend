# 💻 E-Learning Programming App (Backend)

This project is a **Laravel backend system and admin dashboard** developed to support a Flutter-based mobile application for learning programming.

It provides APIs for managing courses, tests, user progress, and integrates AI-based interaction to assist users with programming questions.

---

## ⚙️ Tech Stack

- **Backend:** Laravel  
- **Dashboard:** Bootstrap (Blade Templates)  
- **Mobile App:** Flutter  
- **AI Integration:** Smart assistant for answering programming questions  

---

## 👥 User Roles

### 🔹 User
- Register using name, email, and password  
- Login securely  
- Edit profile  
- Browse and enroll in programming courses  
- Take tests to evaluate learning progress  
- View test results and level  
- Add courses to favorites  
- Ask programming questions via AI assistant  
- Track progress in enrolled courses  
- Participate in competitions  
- View top 5 monthly winners  
- Logout  

---

### 🔹 Admin
- Login using email and password  
- Edit profile  
- Manage courses (Create, Update, Delete)  
- Manage tests (Create, Update, Delete)  
- Manage users (View & Delete accounts)  
- Logout  

---

## 🤖 AI Features

- Users can ask questions about:
  - Programming languages  
  - Coding problems  
- AI provides instant responses to support learning  

---

## 🏆 Competitions

- Users can participate in programming competitions  
- System displays **Top 5 winners monthly**  

---

## 🎯 Features

- RESTful API for Flutter application  
- Role-based authentication  
- Course & test management system  
- Progress tracking system  
- Favorites system  
- AI-powered Q&A assistant  
- Competition and ranking system  
- Admin dashboard for full control  

---

## ⚡ Non-Functional Requirements

### 🔹 Performance
- System responses should not exceed **3 seconds**

### 🔹 Security
- Only admins can manage users, courses, and tests  
- User data is protected and access-controlled  

### 🔹 Scalability
- Supports multiple users simultaneously  

### 🔹 Usability
- Simple, clear, and user-friendly interfaces  

### 🔹 Reliability
- Courses, tests, and results are accurate and consistent
